* 재해연보 - 소방방재청 : 수해로 침수 피해를 입은 시.군.구
* 전국재해위험지구 - 공공데이터활용센터 : 수해로 인해 위험지역으로 지정된 지역
* 홍수 예보 API - 한강홍수통제소 : 실시간 홍수 예보 데이터
* 동네 예보 API - 기상청 : 지역 별 실시간 강수량 예보 데이터
